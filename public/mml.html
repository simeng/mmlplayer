• t = tempo (default 100)
  • o = octave (default o5)
  • v = volume
  • l = change default note length (default 4)
  • r = rest note
    • 1, 2, 4, 8, 16 = note length
    • . = dotted (150%)
  • cdefgab = note (can append half note / length modifiers)
    • + = half note up (#)
    • - = half note down (b)
    • 1, 2, 4, 8, 16 = note length
    • . = dotted (150%)
  • > = 1 octave up
  • < = 1 octave down
  • , = start next track (plays simultaneously)
  • & = combine with next note
